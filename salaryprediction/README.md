# salaryprediction
Salary Prediction Project (Python)

Goal of the project is predictic the salary based on a set of features.

This repository have 2 notebooks, **SalaryPredictionNotebook_EDA** is for analyze and explore the data, **SalaryPredictionNotebook_DEV** is for model development and predict the values. Also have a presentation, that aims to explain and demonstrate better the process and the results.

The files in the data folder are the dataset used in the project and file **predicitions.csv** are the preditions.

The Project Presentation, the pptx file, is under development.
